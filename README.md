# Genetic algorithm practice

Derived from AI Junkie's [Genetics Algorithms in Plain English](http://www.ai-junkie.com/ga/intro/gat1.html).

## Requirements:
* Python 2.7
* Python 3.5.1

## To run:
`python3 gen.py`

## Testing
`python -m unittest discover`
